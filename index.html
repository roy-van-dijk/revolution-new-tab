Show Quick Tips
  • To change links, images, and tile colours, turn on edit mode.
  • For tile icons, try using images with transparent backgrounds for the best results. You can identify these on Google Image Seach via the checkerboard pattern when previewing them. Example.
  • For all images, but especially the background image, try using images that aren't too large in filesize or are delivered by slow servers. This may harm page performance or load times.
